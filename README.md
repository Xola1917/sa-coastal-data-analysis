# SA Coastal Data Analysis

Exploratory analysis of South African coastal monitoring data 
from SAEON (South African Environmental Observation Network).

## Objectives
- Identify temperature patterns and seasonal trends
- Detect anomalies in coastal monitoring data
- Visualise findings for environmental reporting

## Tools & Technologies
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebooks

## Data Source
SAEON — saeon.ac.za
